Méthode de Paiement


1. Saisir un numéro de téléphone ayant un compte Mobicash

2. Sélectionnez le boutton VÉRIFICATION DU NUMÉRO ci-dessous

3. Vous recevrez un SMS vous demandant de proceder au paiement de 800 FCFA.

4. Confirmer la transaction en composant le code USSD *555*6*1# .

5. Confirmez par votre code PIN.

6. Sélectionnez le boutton VÉRIFICATION DU PAIEMENT ci-dessous
